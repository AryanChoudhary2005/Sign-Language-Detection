# Real-Time Sign Language Detection System

Developed a real-time Sign Language Detection System using Computer Vision and Deep Learning to recognize hand gestures through a webcam and translate them into meaningful words. The project aims to bridge the communication gap between individuals using sign language and those unfamiliar with it by providing instant gesture recognition.

### Key Features

* Real-time hand gesture detection using a webcam
* AI-powered sign language recognition
* Live prediction and display of recognized gestures
* Automatic hand tracking and image preprocessing
* User-friendly interface for instant interaction

### Technologies Used

**Python, OpenCV, CVZone, MediaPipe, TensorFlow/Keras, NumPy, Computer Vision, Deep Learning**

### Project Highlights

* Implemented real-time hand detection and tracking using the CVZone Hand Tracking Module built on MediaPipe.
* Preprocessed captured hand images by cropping, resizing, normalization, and centering them on a fixed-size canvas before prediction.
* Trained and integrated a TensorFlow/Keras classification model to recognize seven common sign language gestures.
* Performed real-time inference using the trained model and displayed predictions directly on the video stream.
* Built an efficient image processing pipeline to improve recognition accuracy under different hand positions and scales.

### Recognized Gestures

* Hello
* Sorry
* No
* Okay
* Please
* Thank You
* Yes

### Workflow

1. Capture live video from the webcam.
2. Detect and track the user's hand.
3. Crop and preprocess the hand region.
4. Pass the processed image to the trained deep learning model.
5. Predict the corresponding sign language gesture.
6. Display the predicted word in real time on the video feed.

### Outcome

The project successfully demonstrates how Computer Vision and Deep Learning can be combined to perform accurate real-time sign language recognition. It enhanced my practical skills in image preprocessing, hand tracking, TensorFlow model integration, OpenCV, and deploying AI models for real-world human-computer interaction applications.
